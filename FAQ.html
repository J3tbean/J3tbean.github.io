My server is super laggy! What do I do?

This is the most common, yet trickiest question to answer. It can stem from many things, such as:

  • You dont have enough RAM. Refer to the systems requirements page for numbers!
  • Too many players for your system requirements.
  • Outdated server software.
  • Running other programs that use system resources.
  • Your connection isnt sufficient. Slow internet on the server side causes everyone to lag.



What if I want plugins or mods?

This can make things tricky, but fear not! Some small technical know how and youll be there in no time. To install a modded server, you should download the Minecraft Forge server client.
Then, repeat the steps seen on the main page. Now you will have a mod folder to put your mods into!

Plugins can be a little tricky as well. The process is similar thankfully! You have to install a server software called Paper instead of Forge, repeating the same steps as before.

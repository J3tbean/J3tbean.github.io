This website is here to teach you how to set up and run your own minecraft server, both vanilla and modded!
This will also provide resources on how to port forward, what the recommended hardware is, and even help you configure your server configurations!

Where do I begin?

The fated question! This is always the hardest part for a beginner. How do you install the necessary programs for a server?
Its actually rather simple! Generally, you should go to the official website for the .jar file necessary to run it. There are special cases, which we will get into later, but we will do the basics first.

Here is the link to the official Minecraft.net website to download the server.

On this website, there is a download link. At the time of writing, the version is 1.21.5. This is the latest version of Minecraft, and the one you want the most.
Then, you want to put it into its own folder (I recommend placing this folder on the desktop, personally) and run the server.jar file you downloaded.

A bunch of files appeared and my server failed!

No need to panic yet! This part is simple. One of the files that appeared in your new folder is a txt file called EULA.
This is the End User License Agreement for Minecraft, and all you have to do is click into this text file and change the false to true. This means you agree to Minecrafts End User License Agreement.

Now you have all you need to start up your server. But first, we should get into some more technical stuff that you need to know before your friends can play.

Here is the system requirements page!

With that done, we can start our Minecraft Server for the first time. Just start up the server.jar file again, and it will generate the world and start the server!
Now, if your friends are all local with you, great! Your good to go! But most likely, your friends are away from your local network. This means you will have to Port Forward your router. I have provided a youtube link here to learn more, as its rather complicated.
